# Komplete Geek – Free Phone Course for Seniors

A simple, friendly, and 100% free web-based course designed to help seniors learn how to use their Android or iPhone devices.

## Features
* **20 Lessons:** Covering basics, calls, texting, apps, and safety.
* **No Passwords:** Users just enter a nickname to start.
* **Device Toggle:** Switch between Android and iPhone instructions.
* **Interactive Quizzes:** Quick checks to reinforce learning.

## How to use
Simply open `index.html` in any web browser or visit the live link via GitHub Pages.

## Built With
* HTML5, CSS3, and Vanilla JavaScript.
* Designed with assistance from Claude AI.
